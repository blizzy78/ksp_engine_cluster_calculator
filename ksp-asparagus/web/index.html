Engine Configuration Calculator for KSP

by blizzy, based on Temstar's rocket configuration guide

This calculator can be used to work out the engine layout for a central stack and attached booster stacks required for liftoff. Note that booster stacks are expected to be staged "asparagus"-style. Also, to get better results you should allow part clipping because more engines can be put on the fuel tanks.

This page uses advanced HTML and JavaScript features, so better get a modern web browser. Most specifically, Internet Explorer 9 or older are not supported.
